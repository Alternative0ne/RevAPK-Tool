## How to Use ? 

* `git clone https://github.com/Alternative0ne/RevAPK-Tool.git`
* `cd RevAPK-Tool`
* `chmod +x RevAPK-Tool.sh`
* `./RevAPK-Tool.sh`
* To Reverse Android , Type `1` and put the path of apk
* To Reverse IPA , Type `2` and put the path of ipa
* To Exit , Type `3`
